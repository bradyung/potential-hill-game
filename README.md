# Potential Hill

Potential Hill is a small browser game I created for SDL 6 based on Unit 6 topics like potential functions, gradients, and vector fields.

In the game, the player moves through a colorful contour-style field and tries to reach the true highest point. The terrain represents a potential function, and the vector arrows show the direction of greatest increase. Some levels include extra challenges like false summits, valleys, weaker hints, and optional stars for bonus score.

## Features

- 5 short levels with increasing difficulty
- Contour-style terrain visualization
- Dense vector field arrows
- Gradient-based gameplay
- False summits and valley traps
- Hint system
- Score, medals, and star collectibles
- Runs in a web browser on Linux or Windows

## Math Connection

This project connects to Unit 6 by showing how potential functions and gradients work in a visual and interactive way.

- Brighter / warmer areas represent higher potential
- Cooler areas represent lower potential
- The vector arrows point uphill in the direction of steepest increase
- The goal is to use this information to find the maximum point

## How to Run

1. Download or save the file `potential_hill_game.html`
2. Open it in Firefox or another modern browser
3. Play using your keyboard

## Controls

- **WASD** or **Arrow Keys** = move
- **Enter** = start game
- **R** = restart level
- **H** = use hint
- **N** = next level after winning

## Why I Made It

My original SDL 6 idea was to continue improving my SDL 4 Gradient Climber project. After losing Windows while experimenting with Linux, I decided to create a new project that still connected to a similar idea, but would run easily on both Linux and Windows through the browser.
